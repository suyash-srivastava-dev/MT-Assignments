Assignment-05 NMT English-Hindi

Develop English to Hindi translation system using at least two different NMT architecture. Use different configurations of these systems. Include discussion about the models as well.
Analyze the performance using various evaluation metrics.


- Use datasets developed during earlier assignments. Study the following characteristics for datasets
- Correlation between English and Hindi sentence lengths
- Number of word token in English and Hindi (with and without punctuations)
- Word types in English and Hindi
- Number of unique words in English and Hindi (annotated as ÜNK")
- Number of one-to-many translations
- Number of many to one translations
